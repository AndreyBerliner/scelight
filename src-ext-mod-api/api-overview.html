This documentation is the public Scelight External Module API documentation, home page: https://sites.google.com/site/scelight/.

This API documentation is public and is intended for / allowed to be used by anyone.

External Module API entry point:

  • {@link hu.scelightapi.IExternalModule}

Main exposed components: {@link hu.scelightapi.IServices}

  • Application logger: {@link hu.scelightapibase.service.log.ILogger}
  • Language and locale specific utilities: {@link hu.scelightapibase.service.lang.ILanguage}
  • Extended and custom swing GUI component library: {@link hu.scelightapi.service.IGuiFactory}
  • Settings. External modules can specify settings and integrate into the Settings dialog; take advantage of setting dependent and bound components and listen to setting changes. {@link hu.scelightapi.gui.setting.ISettingsUtils}, {@link hu.scelightapi.gui.setting.ISettingsGui}, {@link hu.scelightapi.IModEnv#initSettingsBean(String, java.util.List)}
  • Replay parser engine: {@link hu.scelightapi.sc2.rep.factory.IRepParserEngine}
  • Replay processor engine: {@link hu.scelightapi.sc2.rep.repproc.IRepProcessor}
  • Basic SC2 Balance Data: {@link hu.scelightapi.sc2.balancedata.IBdUtil}, {@link hu.scelightapi.sc2.balancedata.IBalanceData}
  • Name template engine: {@link hu.scelightapi.template.ITemplateEngine}
  • Replay search engine: {@link hu.scelightapi.search.IRepSearchEngine}
  • SC2 monitor (current game status, live APM, game status change listening): {@link hu.scelightapi.service.sc2monitor.ISc2Monitor}
  • Replay folder monitor (new replay listening): {@link hu.scelightapi.service.repfoldermonitor.IRepFolderMonitor}
  • Sound utilities (background sound player - including MP3 format): {@link hu.scelightapi.service.sound.ISound}
  • General utilities to help implement any kind of code: {@link hu.scelightapi.util.IUtils}, {@link hu.scelightapi.util.gui.IGuiUtils}, {@link hu.scelightapi.service.IFactory}

You can find Scelight here: Scelight home page
