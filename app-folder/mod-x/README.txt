Place of External Modules.

Enabled Official External Modules are automatically installed into this folder.

You can extract Unofficial External Modules manually here.
Newly added external modules are disabled by default.
You have to manually enable them on the Installed Modules page inside Scelight.

ONLY INSTALL EXTERNAL MODULES THAT COME FROM TRUSTED SOURCES!

For possible risks please read the help inside Scelight on the Installed Modules page.

You can read more about external modules here:
https://sites.google.com/site/scelight/external-modules
