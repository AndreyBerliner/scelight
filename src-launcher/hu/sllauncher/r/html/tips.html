General ${appName}™ Tips

  1. You can get rid of the mandatory startup delay and all other limitations by registering. For details visit the Registration info page.
    Registration is a one-time action, it does not expire, you don't have to renew it.
  2. On Windows Operating Systems if you can't or don't want to install Java, you can download a prepared JRE (Java Runtime Environment) from the ${appName} Downloads page and extract it into the ${appName} folder. Doing so will allow you to run ${appName} without having to install Java on your system. You only have to do this once.
    The recommended way is still to install Java! Think of this as a last resort.
  3. By default players are listed as they appear in replays. Add yourself to the Favored Player List, and you (and your team) will always be listed and displayed first.
    To do that, click on the Player menu in the Replay Analyzer, and choose Add to the Favored player list. You can also edit the Favored Player List manually on the Replay Processor settings page.
  4. You can pass replays as parameters when you start ${appName}, and they will be opened in Replay analyzers.
    If an instance of ${appName} is already running using the same Workspace, parameters will be passed on to it, and they will be opened in the running instance (which will also be focused / selected).
    Recommended to associate *.SC2Replay files with Scelight.exe on Windows OS, that way if you double click on an SC2 replay file, it will be opened in ${appName}.
  5. If you have multiple accounts and / or you play on multiple regions, you can specify all your accounts as Merged Accounts and that way you will see all your statistics summarized under one—arbitrarily chosen—main account in the Multi-Replay Analyzer. You can specify all your accounts on the Merged Accounts settings page.
  6. The 32-bit Java runs on both 32 and 64-bit Operating Systems and uses much less memory (could be even 70% less)!
    If memory is an issue, it is recommended to install and use the 32-bit Java (uninstall the 64-bit Java if you have both).
