Rows listed in the table can be filtered using the filter components above the table.

You can enter a filter term to the filter text fields, and the listed rows will be filtered , narrowed down immediately.

There are 2 kinds of table filters:
  • Filter: this is an include filter, used to specify what rows you want to include (to be listed).
  • Filter Out this is an exclude filter, used to specify what rows you want to exclude (not to be listed).

When the include filter is active , only the rows that match the filter term will be listed; when the exclude filter is active , only the rows that do not match the filter term will be listed. Green background will notify you if a filter is active and that the listed rows are filtered and you might see an incomplete row list.

Note: The Filter Out filter has higher precedence: if a row matches this filter term, it will not be listed even if it also matches the include filter term.

The filter term is a list of words, and is case insensitive. By default these words are in logical AND connection: a table row matches the filter term if all the words are contained in the row. The word 'AND' is not required but can be put between words (e.g. these 2 terms are equivalent: 'Probe Drone' = 'Probe and Drone' ).
If you put 'OR' between words (or group of words), they will be in logical OR connection: a table row matches the filter term if any of the words (or group of words) is contained in the row.
To filter by an exact phrase which contains spaces or the words 'or' 'and', put it in quotes, e.g. "warp in probe" .

You can clear the filters by clearing the filter terms , or clicking on the cross icon next to the filter text fields, or by pressing the [ESC] key.

Tip: You can focus the include filter text field by pressing [CTRL+F] , and focus the exclude filter text field by pressing [CTRL+T] .
