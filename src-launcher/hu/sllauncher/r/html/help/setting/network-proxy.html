In computer networks, a proxy server is a server (a computer system or an application) that acts as an intermediary for requests from clients seeking resources from other servers.

You may configure to use a Proxy server to connect to the internet if your computer cannot directly reach the internet or the ${appOpName} .

The HTTP Proxy is used when estabilishing a non-secure connection to an HTTP (web) server.
The HTTPS Proxy is used when estabilishing a secure connection to an HTTPS (secure web) server.
The SOCKS Proxy is used when estabilishing a connection not covered by the aforementioned 2 proxies. This is a TCP/UDP level proxy.

Note: ${appName} always uses a secure connection to contact the ${appOpName} which means the HTTPS Proxy applies in this case.
