Helps and tips are styled, formatted content. The following styles are used:

Keywords or important words:  keyword
Terms or names from ${appName}:  term
Keys or shortcut key combos:  [KEY]
Not important but interesting:  not important
Details for advanced users:  advanced details

Some blocks have special meaning or role. They are the following:

This is a definition block.
Usually defines a term, process or expression of ${appName}.

This is a note block.
Usually contains something which is good to know.

This is a tip block.
Gives hints and tips to use ${appName} more efficiently.

This is an example block.
Provides concrete examples with explanation.

This is a warning or very important block.
You should always read these and comply / act accordingly.
