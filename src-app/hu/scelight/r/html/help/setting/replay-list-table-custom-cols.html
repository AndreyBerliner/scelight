Each column offered to you in the Replay list table displays a well defined property of the replays (e.g. the Version column displays the version of the replays).

With Custom columns you have the possibility to tell what properties of the replays you want to display and in what format.

A Custom column is a column which you can name as you like, and you can define its value in the form of a flexible template.

The template of a Custom column will be evaluated by the Template engine for each replay, and the result file name will be displayed (without folder name and extension) under the Custom column .

Note: Custom columns can be made visible or hidden on the Replay List Column Setup dialog.
