GAME EVENTS and TRACKER EVENTS nodes contain lots of child nodes. Expanding all of them recursively could take even several minutes (and usually expanding fully is not even intended / wanted).

To avoid this, you can set the max expand depth of these 2 big nodes. A value of 0 will not expand GAME EVENTS and TRACKER EVENTS . A value of 1 will only show their children , a value of 2 only their grandchildren etc.

Note: This depth only applies when you expand them with the Expand Selected or Expand All buttons. Manually you will still be able to expand to any depth.
