There are 2 kinds of time measurements.

The one we all know and use in our real life is referred to as real-time. In StarCraft II Blizzard uses a different time measurement referred to as game-time.

Note that game-time is identical to real-time in Legacy of the Void expansion, but it differs in Wings of Liberty and in Heart of the Swarm expansions.

Game-time depends on the game speed and is equivalent to the real-time on Normal game speed and is approximately 1.38 times faster compared to the real time on Faster game speed (which means 1 minute and 24 seconds game-time is equal to 1 minute real-time).

Blizzard only shows you game length, time charts and build order information in game-time .

Pros of game-time is that it does not depend on the game speed, a build order will contain the same times regardless of the game speed.
Cons of game-time is that it is simply unnatural and creates confusion.

Since all ladder matches are to be played on Faster game speed, and the number of games played on non-faster game speed is negligible, the pros of game-time quickly fade away. The use of real-time is recommended.
