If enabled, rows in the Replay list table will be colored based on the mach result of the first player found from your Favored Player List .

Based on the match result of this player, if the result is:
  • Victory: the row will be colored to green
  • Defeat: the row will be colored to red
  • Tie: the row will be colored to blue

If no favored players found in the replay, the row will not be colored.
