When enabled in the settings, you can move the main tool bar by simply dragging it with your mouse.

You can drag and drop the tool bar on the four sides of the main window: North , East , South and West . While dragging it, you can also release it anywhere else in which case the Tool bar will be detached and will float above the main window.

The floating Tool bar can be attached back to the main window by simply closing it with the red X icon or dragging and releasing it above one of the four sides of the main window.
