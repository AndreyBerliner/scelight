If you are using a public email service (e.g. not a server run by you), you can only specify your real email address as the sender email address else the SMTP server will either ignore the specified sender address or reject sending the email completely.

Some email services allow connecting multiple email accounts. If this applies to you, you can obviously provide any of those connected / authenticated email addresses.

Email addresses are in the form of
user@host.domain
or
"Personal Name" <user@host.domain>

Note: It is not required but recommended to put the personal name in quotes.
