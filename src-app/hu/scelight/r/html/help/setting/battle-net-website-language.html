The Battle.net websites usually support multiple languages, and websites of different regions support a different set of languages.

You can choose multiple preferred languages. When opening a Battle.net page (e.g. viewing the profile page of a player), ${appName} will choose the first of your preferred languages that is supported on the website to show the page. If none of your preferred languages are supported, a region-specific default language will be selected.
