Regular expressions are a way to describe a set of strings based on common characteristics shared by each string in the set.
A regexp is a sequence of characters that forms a search pattern.

You must learn a specific syntax to create regular expressions which is beyond the scope of this help. The link leads you to the syntax specification.

Example #1: Replays that have a Silver league player on both teams:
League matchup MATCHES '(.*)s(.*)v(.*)s(.*)'
Note: this example is doable without regexp:
[Player A]League=Silver AND [Player A]Team=1 AND [Player B]League=Silver AND [Player B]Team=2
Example #2: Replays whose map name contains a number:
Map name MATCHES '.*\d(.*)'
Example #3: Replays whose map name contains non-alphanumeric, non-space characters (e.g. dash, apostrophe, parenthesis etc.):
Map name MATCHES '.*[^\w\s](.*)'
