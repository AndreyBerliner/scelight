You can also have only a part of the dynamic value of any symbol inserted.

To do so, before the closing square brackets of the symbol insert the wanted character range (first and last character index, starting at 1, both inclusive) wrapped in braces in the form of {first-last} .

Example: To insert the substring of the map name starting from the 2nd character up to the 10th use the following expression:
[map{2-10}]
Which in case of "Cloud Kingdom LE" results in "loud King" .
